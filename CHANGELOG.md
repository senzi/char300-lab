# Changelog

## 0.2.3 - 2026-07-06

### 体验细节优化

- 优化夜间模式下的标识与分享图片表现。
- 调整总览分享图版式与留白。

## 0.2.2 - 2026-07-06

### 离线可用

- 新增 PWA 离线支持，首次在线打开后可在断网时继续写作。
- 缓存应用资源、字体样式和已加载字体分片，后续版本上线后会自动接管并刷新。

## 0.2.1 - 2026-07-06

### 文案与分享卡片优化

- 年度目标改为“先坚持一年”，并按已写天数提示下一站。
- 文章分享卡片底部新增“逐字”文案和访问地址。

## 0.2.0 - 2026-07-06

### 本地存储升级

- 新增 OPFS 本地存储，支持更大的写作档案。
- 升级前会自动下载完整数据 ZIP 备份。
- 旧版 ZIP 导入会继续兼容。
- 右下角新增 OPFS 状态灯，绿灯表示 OPFS 已启用，红灯表示仍在兼容存储或 OPFS 不可用。
- 页脚新增“更新日志”入口。

## 0.1.1 - 2026-07-06

### 夜间模式与总览视图

- 新增夜间模式切换，支持跟随系统、日间、夜间。
- 新增总览视图，展示年度格、写作天数、缺席天数和每日数据。
